PhotoSort
=========

sorts pictures into directories based on tag in the metadata. 
Uses the metadata extractor library.
The remainder of the source code is all authored by Barrett Hinson.

=========
INSTRUCTIONS
=========
create a directory next to the exicutible jar file called "unsorted"
place place picures(.jpg) into the unsorted directory and then run the jar.
all of the pictures will be moved and sorted into a new(if it does not already exist) directory called "sorted"
